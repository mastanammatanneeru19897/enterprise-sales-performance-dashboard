# Enterprise Sales Performance Dashboard

## 📌 Project Overview
An end-to-end Business Intelligence solution designed to track corporate revenue growth, pipeline velocity, and sales representative efficiency. This project bridges the gap between raw database infrastructure and executive decision-making.

## 🛠️ Tech Stack
- **Database:** PostgreSQL (Data Modeling & Advanced SQL Queries)
- **BI Tool:** Power BI (Data Visualization & DAX)
- **Data Source:** Excel/CSV datasets

## 📊 Dashboard Preview
![Sales Dashboard](./images/Amazon Sales Dashboard.png)

## 💡 Key Insights Captured
- **Revenue Velocity:** Real-time tracking of Month-over-Month (MoM) and Year-over-Year (YoY) sales growth.
- **Regional Breakdown:** Identification of underperforming territories and high-margin regions.
- **Sales Rep Performance:** Tracking individual sales performance against quarterly targets.

## 🚀 How to Replicate This Project
1. Clone this repository.
2. Run the SQL scripts inside the `sql_scripts/` folder on your local PostgreSQL instance.
3. Open the Power BI `.pbix` file and change the data source settings to connect to your PostgreSQL server.
